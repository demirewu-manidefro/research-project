Intelligent YouTube Sentiment, Emotion, and Engagement Analysis System

Transforming raw YouTube comment data into actionable business intelligence through Deep Learning and Google Gemini AI.

Explore Documentation

1. Overview

This system is a high-performance web application designed to transform raw YouTube comment data into actionable business intelligence. By leveraging a specialized Deep Learning model for Amharic and Google Gemini AI, the system provides content creators with deep insights into audience behavior, emotional trends, and content virality.

Unlike standard tools, this system is optimized for "low-resource" linguistic environments and offers real-time monitoring to detect audience shifts as they happen.

2. System Functionalities

2.1 Sentiment Analysis

At the core of the system is a Bidirectional LSTM neural network classifying Amharic and English comments into:

  • Positive Praise, support.
  • Negative Criticism, dissatisfaction.
  • Neutral Inquiries, factual statements.

2.2 Emotion Detection

Goes beyond polarity to identify Joy, Anger, and Sadness using complex emoji-based rules and text-pattern matching tuned for Ethiopian expressions.

2.3 Real-Time Monitoring

Dynamic fetching mechanism pulls the latest comments via the YouTube Data API. Results update seamlessly without refreshing.

2.4 Virality Prediction

Calculates a Success Probability Score based on sentiment velocity, comment-to-view ratios, and engagement spikes.

2.5 Intelligent Alert System

Proactive warnings when negative sentiment exceeds 40% or sudden "toxicity" spikes are detected.

2.6 Multi-Video Comparison

Strategic benchmarking tool that ranks content using the Sentiment Performance Index (SPI), identifying top Content Pillars.

2.7 Smart Insights (Google Gemini)

Integrates Google Gemini Pro to provide executive summaries. Gemini analyzes clusters of comments identified by the LSTM model and generates professional recommendations in Amharic.

2.8 Trial-Based Access System

Tiered access control ensuring sustainable resource management. Free users are limited to a set number of analyses with subscription upgrade flows in the backend.

3. Tools and Technologies

Frontend (UI/UX)

React.js (HTML5, CSS3, JavaScript ES6+). A modern, responsive interface for high-speed data visualization.

Backend (API)

FastAPI. High-performance Python framework for asynchronous requests, model inference, and routing.

Intelligence Engine

TensorFlow/Keras (Bidirectional LSTM). Custom-trained for Amharic sentiment detection.

Generative AI

Google Gemini API. Multilingual summarization and strategic recommendations.

Data & External Data

Pandas & NumPy for data processing. YouTube Data API v3 for ingestion.

Visualization

Chart.js / Recharts integrated within React.

4. System Architecture

01

User Layer

User enters a YouTube URL into the React frontend.

02

Request & Extraction Layer

React sends an asynchronous request to the FastAPI backend, which triggers the YouTube Data API.

03

Processing Layer

Text cleaned and tokenized. LSTM predicts sentiment. Rule-based engines detect emotions and predict virality.

04

Intelligence & Response Layer

Gemini AI generates summary. Structured JSON sent back to React for visualization.

5. Methodology: Hybrid Intelligence

One of the system's most professional features is its Hybrid Architecture:

Feature Methodology Reason for Choice
Sentiment Analysis Deep Learning (LSTM) Handles complex linguistic context and grammar of Amharic over basic lexicons.
Emotion Detection Rule-Based (Emojis) Faster than ML; emojis provide 90% accuracy in emotional intent on social media.
Alerts/Virality Mathematical Formulas Rule-based logic provides "Explainable AI" that content creators can trust.
Summarization Generative AI (Gemini) Best-in-class performance for Amharic translation and creative reasoning.
